1. Menjalankan Jekyll

	```bash
	jekyll s
	```

2. Menjalankan Jekyll sambil meng-compile folder `_drafts`

	```bash
	jekyll s --draft
	```

3. Menjalankan Jekyll dengan port yang berbeda

	Misalnya aja kita mau menjalankan dengan port `1990`:

	```bash
	jekyll s --port=1990
	```
